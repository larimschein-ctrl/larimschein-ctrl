# 📊 Projetos de Portfólio

## 🔹 Pipeline de Dados em AWS (Telegram)

Projeto de engenharia de dados para ingestão e processamento de dados coletados automaticamente por um bot do Telegram.

**Tecnologias:** AWS, Python, Data Lake, SQL

**Principais etapas do projeto**

* Coleta automatizada de dados via bot do Telegram
* Estruturação e armazenamento dos dados em formato JSON
* Armazenamento em Data Lake no Amazon S3
* Processamento e organização dos dados utilizando AWS Lambda
* Criação de tabela externa no Amazon Athena
* Realização de consultas analíticas em SQL

📎 Projeto:
https://github.com/larimschein-ctrl/telegram-data-pipeline-aws
